Gas Detection System

The Gas Detection System is a safety device designed to detect the presence of gas in a specific area and alert users through various means. It features an LCD interface for user interaction, allowing users to register a phone number for alerts, view the project website, and check the registered number. The system includes a gas sensor (MQ2) that triggers an alarm (buzzer and LED) and sends an SMS and call to the registered number when gas is detected. A reset button allows the user to reset the system after resolving the issue. This project enhances safety by providing timely alerts and facilitating quick responses to potential gas leaks.

The Manual in English

To use the Gas Detection System, turn it on and follow the instructions on the LCD display. First, press 'A' on the keypad to enter your phone number and to delete a char enter '#', then confirm with 'D'. To view the project website name and the App name, press 'B', and to display the registered number, press 'C'. If the MQ2 gas sensor detects gas, the buzzer will sound, the red LED will light up, an alert will appear on the LCD, and an SMS and call will be sent to the registered number. Once the gas issue is resolved, press the push button to reset the system and resume normal operation. Regularly check connections and ensure the communication module is functional for optimal performance.

Le Manuel en français

Pour utiliser le système de détection de gaz, allumez-le et suivez les instructions affichées sur l'écran LCD. Tout d'abord, appuyez sur 'A' sur le clavier pour entrer votre numéro de téléphone pour supprimer un char appuyez sur '#' , puis confirmez avec 'D'. Pour afficher le nom du site web du projet et l'application, appuyez sur 'B', et pour afficher le numéro enregistré, appuyez sur 'C'. Si le capteur de gaz MQ2 détecte du gaz, le buzzer retentira, la LED rouge s'allumera, une alerte apparaîtra sur l'écran LCD et un SMS et un appel seront envoyés au numéro enregistré. Une fois le problème de gaz résolu, appuyez sur le bouton poussoir pour réinitialiser le système et reprendre le fonctionnement normal. Vérifiez régulièrement les connexions et assurez-vous que le module de communication fonctionne correctement pour une performance optimale.
